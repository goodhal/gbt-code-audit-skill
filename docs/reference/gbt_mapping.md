## 映射规则

1. **格式要求**：`GB/T{标准号}-{章节号} {规则名称}`（规则号 + 规则名称）
2. **多标准映射**：Java/C/C++/C# 同时映射专用标准 + 通用基线，用分号分隔
   - 示例：`GB/T34944-6.2.3.3 命令注入; GB/T39412-6.1.1.6 命令行注入`
3. **统计规则**：
   - 专用标准统计表 (34943/34944/34946)：只显示专用标准规则
   - 通用标准统计表 (39412)：单独汇总，Java/C/C++/C# 问题已统计到专用标准
   - 详细发现：保留双映射
